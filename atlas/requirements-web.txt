#     pip install -r requirements.txt        # core (terminal)
#     pip install -r requirements-web.txt     # + the web UI
#
#     chainlit run web/app.py -w              # -> http://localhost:8000
#

# of `opentelemetry-instrumentation-*` packages, INCLUDING
# `opentelemetry-instrumentation-ollama`. That is a passive OpenTelemetry *shim*, not
# the Ollama runtime. literalai does a top-level `from traceloop.sdk import Traceloop`,
# so the suite can't be uninstalled without breaking `import chainlit`. Verified the
# ollama shim is DORMANT: it is never imported (`import chainlit` loads zero ollama
# modules). Nothing here runs or depends on Ollama — the fleet's no-Ollama policy holds.
